확정 스펙

OS Ubuntu 20.04 LTS
CPU 2 × 12-core Intel Xeon E5-2650 v4 계열, 48 threads
Memory 128GB DDR4
Storage 1TB SATA SSD for OS, 8TB storage for data
GPU 8 × NVIDIA Tesla P100 PCIe 16GB
Driver / CUDA NVIDIA Driver 535, CUDA 12.2

Training Implications

  • 기본 학습은 single-node 8-GPU DDP로 설계한다: torchrun --nproc_per_node=8.
  • P100은 BF16을 기대하지 않는다. 기본 precision은 FP32 안정성 확인 후 FP16 AMP를 사용한다.
  • 각 GPU VRAM 16GB이므로 DINOv2/DINOv3/SigLIP2 large full fine-tune은 리스크가 크다.
  • 권장 경로는 frozen teacher embedding 생성, lightweight student 학습, 필요 시 마지막 block partial fine-tune이다.
  • 데이터는 8TB storage에 두되, 학습 중 dataloader 병목이 생기면 이미지 resize/cache 또는 LMDB/WebDataset 변환을 검토한다.
  • Driver 535/CUDA 12.2 기준 PyTorch wheel 호환성을 설치 전에 확인한다.
